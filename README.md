Установка nginx из исходных файлов
================

# Необходимые пакеты

Замените версии nginx, openssl и pcre на требуемые в соответствующих папках

Ссылки для скачивания
- [nginx]
- [pcre]
- [openssl]

# Установка

В каталоге nginx запустите следующий набор команд:

`cd nginx
./install.sh
make
sudo make install`
